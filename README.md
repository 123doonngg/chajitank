# 🎮 坦克大战 (Tank Battle)

一个纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏，无需任何依赖，直接在浏览器中打开即可畅玩。

## 🎯 游戏特性

- 🕹️ **经典玩法** — 控制坦克在战场上移动，消灭敌方坦克
- 🧱 **多种地形** — 砖墙（可摧毁）、钢铁（不可摧毁）、水域、树丛
- 🏆 **护基地** — 保护你的基地不被敌人摧毁
- 📈 **多个关卡** — 3 个预设关卡，通关后循环
- 💎 **道具系统** — 生命恢复、无敌护盾、全屏炸弹、加速
- 🔊 **音效** — 射击、爆炸、道具拾取等音效
- 🎨 **粒子特效** — 爆炸时的粒子效果

## 🕹️ 操作方式

| 按键 | 操作 |
|------|------|
| `W` `A` `S` `D` 或 `↑` `←` `↓` `→` | 移动坦克 |
| `空格` | 发射炮弹 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/YOUR_USERNAME/chajitank.git
```

2. 用浏览器打开 `index.html` 即可开始游戏。

或者直接双击 `index.html` 文件在浏览器中打开。
